# 房间消息概述

## 介绍

* 房间消息是指用户通过调用SDK接口进入房间后，可以在该房间发送和接收广播消息。用户退出房间后不会再接收到该房间的消息。
* 由于房间人数最多可达30万人，在消息数量特别巨大的情况下房间消息不保证百分百到达，但普遍情况下房间消息是不会丢的。如因特殊情况需保证某一类房间消息百分百到达，请联系我们。

## 适用场景

| 领域分类 | 适用功能                                                 |
| -------- | -------------------------------------------------------- |
| 直播     | 弹幕<br>聊天室 <br/>直播间礼物、点赞                     |
| 社交     | 聊天室消息                                               |
| 教育     | 直播教室发言消息<br>权限管理（奖励、上讲台、举手、点赞） |
